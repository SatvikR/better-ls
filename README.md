## Install

```sh
go get github.com/SatvikR/better-ls
```

## Build from source

```sh
# Clone repo
git clone https://github.com/SatvikR/better-ls
cd better-ls

# Compile
go build

# Run (make sure add .exe on windows)
./better-ls
```
